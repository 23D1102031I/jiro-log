Jiro Log 画面遷移仕様書（v2 更新: 2026-04-25）

1. グローバルナビゲーション（共通ヘッダー）
全画面の上部に固定で配置され、主要な5画面への遷移を司ります。

🏠 ホーム -> ホーム画面へ遷移

📍 店舗マップ -> 店舗マップ画面へ遷移

👑 スタンプカード -> スタンプカード画面へ遷移

✏️ レビュー投稿 -> レビュー投稿画面へ遷移

👤 マイページ -> マイページ画面へ遷移

2. 各画面の構成と遷移定義

① ホーム画面 (Home) /
アプリのポータルとなる画面。各機能へのハブとしての役割を持ちます。

画面構成要素:

  最新の神豚投稿（ヒーローバナー）

  最新レビュータイムライン（タブ切り替え）
    - 「すべての最新」タブ: 全ユーザーのレビューを新着順
    - 「フォロー中」タブ: フォロー中ユーザーのレビューのみ（ログイン時のみ）

  直系コンプリートへの道（スタンプカード進捗プレビュー）

  店舗マップ（プレビュー）

  店舗ランキング

  フローティングアクションボタン（レビューを投稿する）

遷移先 (アクション):

  タイムラインのレビューカードをタップ -> /reviews/[id]（レビュー詳細）

  タイムラインの店舗名をタップ -> /stores/[id]（店舗詳細）

  タイムラインのユーザーアイコン/名前をタップ -> /users/[id]（ユーザープロフィール）

  「スタンプカードを見る」ボタン -> /stamp（スタンプカード）

  「店舗マップを開く」ボタン -> /map（店舗マップ）

  FAB「レビューを投稿する」ボタン -> /post（レビュー投稿）

② 店舗マップ画面 (Store Map) /map
現在地周辺や特定のエリアから直系店舗を探す画面です。

画面構成要素:

  検索バー / 絞り込みフィルター（エリア、営業状況）

  店舗リスト（左側カラム）

  インタラクティブマップ（CartoDB Positron タイル・右側エリア）
    - 最小ズームレベル: 5

遷移先 (アクション):

  左側のリストの店舗をタップ -> 地図がその店舗にフォーカス（状態変化）

  マップ上のピンをタップ -> ポップアップを表示し、リストを同店舗に同期

  ポップアップ/リストの「詳細ページへ」 -> /stores/[id]（店舗詳細）

③ スタンプカード画面 (Stamp Card) /stamp
直系全店制覇のモチベーションを高める実績画面です。

画面構成要素:

  全45店舗のスタンプグリッド（「完食」と「未訪問」でステータス表示）

  現在の制覇状況（サークルプログレス、◯/45店）

  獲得した最高位の称号

遷移先 (アクション):

  各店舗のスタンプをタップ -> /stores/[id]（店舗詳細）

④ レビュー投稿画面 (Review Post) /post
食べた一杯の詳細を記録する入力フォーム画面です。

画面構成要素:

  店舗選択（ドロップダウン）

  食べた日（日付ピッカー・デフォルト今日）

  ラーメン画像アップローダー（最大5枚）

  コール（呪文）選択ボタン（ニンニク、ヤサイ、アブラ、カラメ）

  総合評価（★1〜5）

  コメント入力エリア（最大800文字）

  ジロリアンパラメータ（6項目の独自スライダー・各5段階）

遷移先 (アクション):

  「この一杯を記録する」ボタン -> 称号獲得演出（あれば）→ ホーム画面へ遷移、トースト通知を表示

⑤ マイページ画面 (My Page) /mypage
ユーザー自身のプロフィールと実績を振り返る画面です。

画面構成要素:

  ユーザープロフィール（アイコン、ユーザー名・編集可、最高位称号）

  サマリー数値（総実食数、制覇店舗、レビュー数、総いいね数）

  フォロー数・フォロワー数

  My Jiro Identity（6項目の平均値レーダーチャート）

  獲得した称号一覧

  レビュー履歴リスト

  アカウント退会ボタン

遷移先 (アクション):

  レビュー履歴の各項目タップ -> /reviews/[id]（レビュー詳細）

  レビュー履歴の店舗名タップ -> /stores/[id]（店舗詳細）

  「カレンダーを見る」リンク -> /calendar（訪問カレンダー）

⑥ 店舗詳細画面 (Store Detail) /stores/[id]

画面構成要素:

  店舗情報（名前・住所・エリア・定休日・営業時間・SNSリンク）

  店舗平均 My Jiro Identity レーダーチャート

  レビュータイムライン（最新20件・無限スクロール）

遷移先 (アクション):

  レビューカードをタップ -> /reviews/[id]（レビュー詳細）

  レビューのユーザーアイコン/名前をタップ -> /users/[id]（ユーザープロフィール）

⑦ レビュー詳細画面 (Review Detail) /reviews/[id]

画面構成要素:

  投稿者アバター・ユーザー名・最高位称号バッジ

  店舗名・住所

  レビュー画像（複数枚グリッド）

  総合評価・コール・コメント

  My Jiro Identity レーダーチャート + 6パラメータ

  いいね数・シェアボタン（X投稿）

  食べた日（eaten_at）

  同じ店のレビューグリッド（最新6件）

遷移先 (アクション):

  ユーザーアイコン/名前をタップ -> /users/[id]（ユーザープロフィール）

  店舗名をタップ -> /stores/[id]（店舗詳細）

  同じ店のレビューをタップ -> /reviews/[id]（別のレビュー詳細）

⑧ 他ユーザープロフィール画面 (User Profile) /users/[id]

画面構成要素:

  アバター・ユーザー名・最高位称号

  フォロー/フォロー解除ボタン（ログイン時・自分以外に表示）

  フォロー数・フォロワー数・レビュー数

  サマリー（総実食数・制覇店舗・総いいね数）

  My Jiro Identity レーダーチャート

  獲得称号一覧

  レビュー履歴タイムライン

遷移先 (アクション):

  レビューカードをタップ -> /reviews/[id]（レビュー詳細）

  「訪問カレンダーを見る」リンク -> /users/[id]/calendar

⑨ 訪問カレンダー画面 /calendar（自分）/ /users/[id]/calendar（他人）

画面構成要素:

  月別カレンダー（前月/次月ナビ）

  レビューがある日: 🍜アイコンを表示

遷移先 (アクション):

  🍜アイコンをタップ -> その日のレビュー一覧をモーダルまたは展開表示

  レビューをタップ -> /reviews/[id]（レビュー詳細）

3. ルーティング設計（確定版）

/ ホーム
/map 店舗マップ
/stamp スタンプカード
/post レビュー投稿（ログイン必須）
/mypage マイページ（ログイン必須）
/stores/[id] 店舗詳細
/reviews/[id] レビュー詳細
/users/[id] 他ユーザープロフィール
/calendar 自分の訪問カレンダー（ログイン必須）
/users/[id]/calendar 他ユーザーの訪問カレンダー（常に公開）
/admin 管理者ページ（管理者ロールのみ）
/profile/setup 初回プロフィール設定
