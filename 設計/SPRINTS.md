## 各スプリント詳細

### S01: 基盤（レイアウト・ミドルウェア・認証）
**実装内容:**
- グローバルレイアウト（ヘッダー・フッター・ナビゲーション）
- Supabase Middleware（セッション管理）
- Google OAuth ログイン/ログアウト
- 初回ログイン判定 → プロフィール設定画面への強制遷移
- プロフィール設定画面（/profile/setup）
- トースト通知コンポーネント
- ローディングスピナーコンポーネント
- フッター免責事項

**合格基準:**
- [ ] TypeScriptビルドエラーなし
- [ ] ヘッダー・フッターが全画面で表示される
- [ ] Googleログインボタンが機能する
- [ ] ログイン後にusernameが未設定ならプロフィール設定へ遷移する
- [ ] ゲストはレビュー投稿ページにアクセスできない（リダイレクト）
- [ ] フッターに免責事項が表示される

---

### S02: ホーム画面
**実装内容:**
- 最新レビュータイムライン（初回20件）
- 神豚投稿ヒーローバナー
- スタンプカード進捗プレビュー
- 店舗マッププレビュー
- フローティングアクションボタン（レビュー投稿）

**合格基準:**
- [ ] ビルドエラーなし
- [ ] タイムラインにレビューが表示される
- [ ] FABがレビュー投稿ページへ遷移する
- [ ] モバイル表示が崩れない

---

### S03: 店舗マップ
**実装内容:**
- React Leaflet + OpenStreetMap（初期表示: 三田本店）
- 訪問済み/未訪問ピン（色分け）
- 左カラム: 店舗リスト + 検索・フィルター
- 営業ステータスバッジ（JST判定）

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 地図が表示される
- [ ] 三田本店付近を初期表示する
- [ ] ピンをクリックすると店舗情報が表示される
- [ ] 営業ステータスが正しく表示される

---

### S04: レビュー投稿
**実装内容:**
- 店舗選択（ドロップダウン）
- 画像アップロード（最大5枚・5MB制限・クライアント圧縮）
- コール選択（ピル型ボタン）
- 総合評価（★1〜5）
- コメント（800文字制限）
- ジロリアンパラメータスライダー（6項目）
- NGワードフィルター
- 投稿後トースト通知
- 称号リアルタイム判定・獲得演出

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 未ログインはログイン画面へリダイレクト
- [ ] 5MB超の画像をアップロードするとエラーが表示される
- [ ] NGワードを含む投稿が弾かれる
- [ ] 正常に投稿できる
- [ ] 投稿後にトースト通知が表示される

---

### S05: マイページ
**実装内容:**
- プロフィール表示（アイコン・ユーザー名・最高位称号）
- サマリー（総実食数・制覇店舗・レビュー数・総いいね数）
- My Jiro Identity レーダーチャート
- 獲得称号一覧
- レビュー履歴タイムライン
- アカウント退会機能

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 自分のレビュー数が正しく表示される
- [ ] レーダーチャートが描画される
- [ ] 退会ボタン → 確認モーダル → 退会できる

---

### S06: スタンプカード
**実装内容:**
- 全45店舗グリッド
- 訪問済み/未訪問スタンプ表示
- サークル型プログレスバー
- 最高位称号フィーチャー

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 45店舗分のグリッドが表示される
- [ ] 訪問済みがカラー表示になる
- [ ] プログレスが正しく計算される

---

### S07: 称号システム
**実装内容:**
- 全28種類の称号判定ロジック
- レビュー投稿時リアルタイム判定
- 獲得演出（画面中央ポップアップ・複数並列表示）
- user_titlesへの保存

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 1杯投稿後「初陣」称号が付与される
- [ ] 称号獲得時にポップアップが表示される
- [ ] マイページに獲得称号が表示される

---

### S08: 管理者ページ
**実装内容:**
- /admin（管理者のみアクセス）
- 店舗一覧・追加・編集フォーム
- 非管理者はリダイレクト

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 非管理者がアクセスするとリダイレクトされる
- [ ] 管理者は店舗を追加できる

---

### S09: OGP・シェア機能
**実装内容:**
- レビュー詳細ページのOGP（Next.js ImageResponse）
- ラーメン画像 + 星評価 + レーダーチャート合成
- 画像なしの場合はロゴフォールバック

**合格基準:**
- [ ] ビルドエラーなし
- [ ] /reviews/[id]/opengraph-image が画像を返す
- [ ] 星評価が画像に含まれる

---

### S10: 最終調整
**実装内容:**
- 無限スクロール（Intersection Observer）
- いいね機能（トグル・カウント表示）
- 全画面のレスポンシブ確認
- パフォーマンス最終確認

**合格基準:**
- [ ] スクロール時に20件ずつ追加読み込みされる
- [ ] いいねがトグルできる
- [ ] モバイル（375px）・PC（1280px）で全画面が崩れない
