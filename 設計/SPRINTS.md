## 各スプリント詳細

### S01: 基盤（レイアウト・ミドルウェア・認証）
**実装内容:**
- グローバルレイアウト（ヘッダー・フッター・ナビゲーション）
- Supabase Middleware（セッション管理）
- Google OAuth ログイン/ログアウト
- 初回ログイン判定 → プロフィール設定画面への強制遷移
- プロフィール設定画面（/profile/setup）
- トースト通知コンポーネント
- ローディングスピナーコンポーネント
- フッター免責事項

**合格基準:**
- [ ] TypeScriptビルドエラーなし
- [ ] ヘッダー・フッターが全画面で表示される
- [ ] Googleログインボタンが機能する
- [ ] ログイン後にusernameが未設定ならプロフィール設定へ遷移する
- [ ] ゲストはレビュー投稿ページにアクセスできない（リダイレクト）
- [ ] フッターに免責事項が表示される

- 営業ステータスバッジ（JST判定）

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 地図が表示される
- [ ] 三田本店付近を初期表示する
- [ ] ピンをクリックすると店舗情報が表示される
- [ ] 営業ステータスが正しく表示される

### S04: レビュー投稿
**実装内容:**
- 店舗選択（ドロップダウン）
- 画像アップロード（最大5枚・5MB制限・クライアント圧縮）
- コール選択（ピル型ボタン）
- 総合評価（★1〜5）
- コメント（800文字制限）
- ジロリアンパラメータスライダー（6項目）
- NGワードフィルター
- 投稿後トースト通知
- 称号リアルタイム判定・獲得演出

**合格基準:**
- [ ] ビルドエラーなし
- [ ] 未ログインはログイン画面へリダイレクト
- [ ] 5MB超の画像をアップロードするとエラーが表示される
- [ ] NGワードを含む投稿が弾かれる
- [ ] 正常に投稿できる
- [ ] 投稿後にトースト通知が表示される
