Jiro Log - 他人レビュー詳細画面（/reviews/[id]）詳細仕様書
1. 画面概要・目的
他のジロリアンが投稿した個別のレビュー（一杯の記録）を詳細に閲覧する画面。
巨大なシズル感のある写真、選択されたコール（呪文）、6角形レーダーチャートによるパラメータ分析を通じて、その一杯の魅力を最大限に伝える。いいね・Xシェアのリアクション起点としても機能する。

2. 全体レイアウト・UIポリシー
デザインシステム: 背景色は極薄いグレー（#F9FAFB）。情報は rounded-xl shadow-sm border border-gray-100 の白いカードUIに分割して配置する。

PC版レイアウト（ワイド画面）:
grid grid-cols-1 lg:grid-cols-[minmax(0,1fr)_360px] gap-8 items-start
- 左カラム: メインコンテンツ（写真・評価・コール・コメント）
- 右カラム: lg:sticky lg:top-20 でスクロールに追従（幅360px固定）

3. 画面構成要素

3.1. 左カラム（メインレビューエリア）

① 投稿者カード
- ユーザーアバター（w-12 h-12 rounded-full）＋ ユーザー名（font-bold）
- 最高位称号バッジ（bg-[#FFFF00] text-black rounded-full 👑アイコン付き）
- カード全体がユーザープロフィール（/users/[id]）へのリンク

② 店舗情報カード
- 店舗名（text-lg font-black）を店舗詳細（/stores/[id]）へリンク
- 住所（text-xs text-gray-400）
- 右端に投稿日（eaten_at 優先、なければ created_at）

③ メイン画像
- 1枚目: aspect-video（16:9）でフルワイド表示
- 2枚目以降: grid-cols-4 の小サムネイル（aspect-square）で横並び

④ 総合評価カード
- 星5つ（fill-[#FFFF00]）＋ 数値（text-2xl font-black）

⑤ コールカード
- ヤサイ・ニンニク・アブラ・カラメ の4項目を常に表示
- マシ/マシマシ → 黄色背景（bg-[#FFFF00]）黒文字黒ボーダー
- 標準/その他 → 白背景 グレー文字 グレーボーダー
- null の場合は「標準」と表示

⑥ コメントカード
- bg-gray-50 rounded-2xl でコメント全文を whitespace-pre-wrap で表示

⑦ いいね数表示（静的）
- ❤️ アイコン＋「N いいね」の静的テキスト（ページロード時の件数）

3.2. 右カラム（分析＆関連情報エリア - Sticky）

① ジロリアン・パラメータカード（「このジロリアンのパラメータ」）
- RadarChart（SVG・size=200）を中央に表示
- 6つのパラメータをリードオンリーのプログレスバーで表示
  項目: 麺の太さ / デロさ / 野菜量 / 麺量 / 神豚度 / 乳化度
  （各 bg-gray-100 rounded-full h-2、塗り部分 bg-[#FFFF00]、右端に数値）
- スコアが null の場合は 0 として扱う

② レビューへのアクションカード（ReviewDetailActions）
- ❤️ いいね（Supabaseでトグル、件数表示）
- 𝕏 Xで共有（Web Share API → フォールバックでTwitter intent URL）
- ログイン未ログ時はいいねボタンをdisabledに

③ このユーザーの他のレビューカード
- 同ユーザーの他レビュー最大3件を grid-cols-3 のミニカードで表示
- 各カード: aspect-square画像 + 星評価オーバーレイ + 店舗名
- 「このユーザーのすべてのレビューを見る →」リンクで /users/[id] へ遷移

4. レスポンシブ挙動（モバイル対応）
モバイル（lg未満）では2カラム構成を解除し、縦積み（1カラム）のスクロールレイアウトに変更。
表示順序:
1. 投稿者カード
2. 店舗情報カード
3. 大画像（aspect-video）
4. 総合評価カード
5. コールカード
6. コメントカード
7. いいね数
8. ジロリアン・パラメータ（RadarChart + バー）
9. レビューへのアクション（いいね・シェア）
10. このユーザーの他のレビュー

5. アクション・画面遷移
投稿者アイコン/名前タップ: ユーザープロフィール画面（/users/[id]）へ遷移。
店舗名タップ: 店舗詳細画面（/stores/[id]）へ遷移。
「いいね！」ボタン: トグルでいいね付与/解除。ハートが赤に変化。
「Xで共有」ボタン: Xのシェアインテントへ遷移。
「このユーザーの他のレビュー」カードタップ: 別のレビュー詳細画面へ遷移。

6. データ要件
reviews: id, rating, comment, images, created_at, eaten_at, store_id, call_garlic, call_yasai, call_abura, call_karame, thickness_score, dero_score, vegetable_score, noodle_score, pork_score, emulsification_score, users(id, username, avatar_url), stores(id, name, address)
並行取得: user_titles（最高位称号）/ 同ユーザーの他レビュー3件（user_id一致・自身除外）/ likes（いいね数）
